# Roadmap

1. Go monorepo baseline.
2. FT1.2 protocol core.
3. Emulator TCP service.
4. Gateway polling service.
5. gRPC contracts.
6. Web UI.
7. Docker, observability, and CI.
8. Final docs and release polish.

Step 1 is complete when the root module, active commands, shared baseline
packages, legacy migration, docs, and build/test checks are in place.
